# docs/ — 機能要件・アーキテクチャフォルダ

操作者から機能要件の整理・作成を指示された場合に、本フォルダへ格納する。

## ファイル命名規則

```
<機能名>.architecture.md
例: lineItemSorter.architecture.md
    dynamicEditableTable.architecture.md
```
